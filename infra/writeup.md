1. Из SYSTEM hive достаём boot key, из SECURITY hive дампим LSA secrets, получаем dpapi_machinekey и dpapi_userkey.

2. В Protect/S-1-5-18 лежат зашифрованные master keys. Расшифровываем их через dpapi_machinekey/userkey из шага 1. Получаем 6 расшифрованных ключей.

3. Из SOFTWARE hive достаём сертификат - он лежит в Microsoft\SystemCertificates\MY\Certificates. Парсим blob, вытаскиваем DER, конвертируем в PEM. Issuer - MOYKA ROOT CA, subject - IVANOV-NB, RSA 4096.

4. Файл приватного ключа лежит в Crypto/RSA/MachineKeys. Внутри DPAPI blob, который ссылается на master key c6859e4a... Расшифровываем его ключом из шага 2 (AES-256 + SHA-512). Получаем CAPI PRIVATEKEYBLOB с magic RSA2.

5. Из расшифрованного blob-а достаём модуль n, простые p и q (всё LE). Проверяем n = p*q - сходится. Вычисляем закрытую экспоненту d, собираем RSA ключ, экспортируем в PEM.
